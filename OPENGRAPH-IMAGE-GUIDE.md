# Como criar a imagem OpenGraph

Para ter uma imagem OpenGraph perfeita, você precisa criar uma imagem com as seguintes especificações:

## Especificações:
- **Tamanho:** 1200x630 pixels
- **Formato:** JPG ou PNG
- **Nome do arquivo:** `opengraph-image.jpg` ou `og-image.jpg`
- **Local:** Coloque na pasta `public/`

## Design sugerido:
1. **Fundo:** Gradiente azul (#1e2a8a) para laranja (#f97316)
2. **Logo:** Ícone de raio centralizado no topo
3. **Título:** "Eletro Soluções" em fonte grande e bold
4. **Subtítulo:** "Projetos de Energia Solar Fotovoltaica"
5. **Descrição:** "Homologações • Consultoria Técnica • Suporte Completo"

## Ferramentas para criar:
- **Canva:** https://www.canva.com/ (template "Facebook Post" 1200x630)
- **Figma:** https://www.figma.com/
- **Photoshop/GIMP:** Criar do zero

## Depois de criar:
1. Salve como `opengraph-image.jpg`
2. Coloque na pasta `public/`
3. A imagem será automaticamente usada pelo OpenGraph

## Teste a imagem:
- Facebook Debugger: https://developers.facebook.com/tools/debug/
- Twitter Card Validator: https://cards-dev.twitter.com/validator
- LinkedIn Post Inspector: https://www.linkedin.com/post-inspector/
